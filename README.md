# VezoraFeedback

A modern Minecraft Paper plugin for collecting player feedback through an easy-to-use GUI.

## Features

- ⭐ Staff Review System
- 💡 Suggestions
- 🐞 Bug Reports
- 📊 Rating System
- 🛡️ Admin Panel
- 💬 Chat Input System
- 📁 YAML Storage

## Commands

| Command | Description |
|---------|-------------|
| /feedback | Open feedback menu |
| /feedback admin | Open admin panel |
| /feedback reload | Reload plugin |

## Permissions

| Permission | Description |
|------------|-------------|
| vezora.feedback.admin | Access admin panel |

## Requirements

- Paper 1.21+
- Java 21

## Building

```bash
mvn clean package
```

The compiled JAR will be located in:

```
target/VezoraFeedback-x.x.x.jar
```
